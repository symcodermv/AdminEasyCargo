Customer Notification Setup Guide

When you mark an order Delivered, EasyCargo automatically: sends an in-app notification, sends Telegram alert to all admins, and opens WhatsApp with a pre-written message to the customer. Below are 3 ways to set up fully automatic customer messaging.

Option 1 — WhatsApp ACTIVE NOW

When you click "Update Status → Delivered", WhatsApp automatically opens with the customer's Maldives number (+960XXXXXXX) and a pre-filled message. Just tap Send in WhatsApp.

✅ No setup needed — works right now
Option 2 — Telegram Auto-Message to Customer SETUP REQUIRED

To send automatic Telegram messages to customers, each customer must start your bot once. Steps:

1. Open Telegram → search @BotFather → /newbot → get your Bot Token
2. Add a "Connect Telegram" button to your app — customer taps it, starts your bot
3. Bot saves their Telegram chat_id to Firebase when they start it
4. On Delivered, send message to their saved chat_id automatically
⚡ Ask developer to add Telegram connect — takes ~30 minutes
Option 3 — Auto SMS (Twilio) PAID SERVICE

For fully automatic SMS to any Maldives number (no WhatsApp needed):

1. Sign up at twilio.com → get Account SID + Auth Token + Phone Number (~$1/month)
2. Enable Twilio for Maldives (MV) in the console
3. Add a backend function (Firebase Cloud Function) to call Twilio API on Delivered
4. Cost: ~$0.05 per SMS to Maldives
💡 Best for businesses — requires backend/server setup

Setup Guide — Step by Step

1
Create Twilio Account
Go to twilio.com/try-twilio → Sign up free (get $15 trial credit). No credit card needed to start.
2
Get a Phone Number
Twilio Console → Phone Numbers → Buy a Number. Choose US (+1) number — cheapest option, works for sending to Maldives. Cost: ~$1/month.
3
Copy Your Credentials
Twilio Console → Dashboard → copy Account SID and Auth Token. Paste both above and save.
4
Deploy Cloud Function
You need a small backend to call Twilio (browser security blocks direct calls). Copy the Cloud Function code below and deploy it to Firebase Functions.
5
Test & Go Live
Use the test form above to send a test SMS to your own number. Once working, every "Delivered" status update will automatically SMS the customer.

Firebase Cloud Function Code

Save this as functions/index.js in your Firebase project. Run firebase deploy --only functions to deploy.

const functions = require("firebase-functions");
const admin = require("firebase-admin");
const twilio = require("twilio");
admin.initializeApp();

exports.sendDeliverySMS = functions.https.onCall(async (data, context) => {
  // Get Twilio credentials from Firebase config
  const cfg = functions.config().twilio;
  const client = twilio(cfg.sid, cfg.token);

  const to = "+960" + data.phone.replace(/[^0-9]/g, "");
  const body = data.message;

  try {
    const msg = await client.messages.create({
      body: body,
      from: cfg.from,
      to: to
    });
    return { success: true, sid: msg.sid };
  } catch (err) {
    throw new functions.https.HttpsError("internal", err.message);
  }
});
Deploy commands (run in terminal):
npm install -g firebase-tools
firebase login
firebase init functions   (select your project: easycargo-9c0cd)
cd functions
npm install twilio
cd ..
firebase functions:config:set twilio.sid="ACxxxx" twilio.token="xxxx" twilio.from="+1415xxxx"
firebase deploy --only functions
